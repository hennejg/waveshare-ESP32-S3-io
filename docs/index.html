Waveshare ESP32-S3-POE-ETH Firmware

Open-source firmware for the ESP32-S3-POE-ETH-8DI-8DO and ESP32-S3-POE-ETH-8DI-8RO industrial I/O modules.

Flash latest release

Full firmware

Rule engine, MQTT, CAN bus, Modbus RTU, full web UI. No Matter.

Install Full Firmware

Requires Chrome or Edge with USB access.

Matter firmware

WiFi provisioning + Matter (Apple Home / Google Home). No rule engine, no MQTT.

Install Matter Firmware

Requires Chrome or Edge with USB access.
