Automation Rules

JavaScript rule script — changes are saved to the device and applied immediately without reboot. Leave empty and save to revert to the built-in demo. See the rule language reference.

DSL reference 
// Subscribe to an MQTT topic:
var t = mqtt('rules/trigger').is(function(m) { return m === 'ON'; });

// Read a digital input state:
var di0 = input(0).is(false);   // true when DI0 is open (not conducting)

// Define a rule:
rule('my-rule')
  .when(t, di0)               // all conditions must be true
  .then(function() {
    output(0).set(true);      // set digital output 0
    print('fired!');          // logs to the ESP console
  });
