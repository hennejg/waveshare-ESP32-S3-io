Device Configuration

Device

Used as DHCP hostname. Alphanumeric and hyphens only.

Status mode: 50% yellow = alive, 50% green = network, 50% purple = MQTT. Red/blue flashes = RX/TX traffic. Takes effect after reboot.

MQTT Broker

Digital Inputs

InputNameInvert

Name: used as MQTT topic fragment instead of the number (max 20 chars, no /, unique). Invert: checked = active-low (GPIO low → true).

Digital Outputs

OutputNameInvert

Name: used as MQTT topic fragment. Invert: checked = physical bit flipped relative to logical state.

CAN Bus

GPIO2 (TX) / GPIO3 (RX). Changes take effect after reboot.

If already taken on the bus, the device will claim the next free address automatically.

Modbus RTU

RS485 on GPIO17/18. Changes take effect after reboot.
